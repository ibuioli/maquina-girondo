# Maquina Girondo
Algortimo de Textos Generativos (Versos y Poemas) con partido estético de Oliverio Girondo. 

### clase Verso:

* Cantidad de Palabras [byte] [1 - 7]
* Estructura sintáctica del verso [byte] [0 - 1]
