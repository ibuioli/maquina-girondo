# Maquina Girondo
Algortimo de Textos Generativos (Versos y Poemas) con partido estético de Oliverio Girondo. 

clase Verso:

Recibe dos parámetros tipo byte:

* Cantidad de Palabras [1 - 7]
* Estructura sintáctica del verso [0 - 1]
